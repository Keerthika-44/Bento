# 🍱 Bento Grid Layout

A modern and responsive Bento Grid layout built using HTML and CSS. This project demonstrates the use of CSS Grid to create visually appealing card-based layouts inspired by modern dashboard and portfolio designs.

## 🚀 Features

- 📱 Fully Responsive Design
- 🎨 Modern Bento Grid Layout
- ⚡ Built with HTML5 & CSS3
- 📐 CSS Grid and Flexbox
- 💻 Clean and Simple UI
- 🌐 Cross-browser Compatible

## 🛠️ Technologies Used

- HTML5
- CSS3
- CSS Grid
- Flexbox

## ▶️ How to Run

1. Clone this repository:
   ```bash
   git clone https://github.com/Keerthika-44
/Bento-Grid.git
   ```

2. Open the project folder.

3. Open `index.html` in your browser.
